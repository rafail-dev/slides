# Asciinema

_Record and replay terminal sessions — stored as plain text_

- 🎬 **Record** once, replay anywhere
- 📝 **Plain text** `.cast` format — diffable, versionable
- ✂️ **Copy-paste** from the player — unlike video
- 🌐 [**Embed**](https://docs.asciinema.org/integrations/) in slides, docs, obsidian or any webpage
- 📡 [**Live stream**](https://docs.asciinema.org/manual/server/streaming/#live-streaming) terminal sessions in real time
- 🖼️ **Export** to GIF via [agg](https://github.com/asciinema/agg), SVG, or MP4

[asciinema.org](https://asciinema.org/)
